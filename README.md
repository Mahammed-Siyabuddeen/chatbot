## Running this project
Clone this Project
```
git clone https://github.com/Mahammed-Siyabuddeen/chatbot/
```
 Frontend installation
```
cd client
npm install
npm start
```
 Backend installation
```
cd server
npm install
npm run dev
```
### env configuration
server folder
```
API_KEY
MODEL_NAME
FRONTEND_URL
PORT
```

client folder
```
REACT_APP_BACKEND_URL
```
